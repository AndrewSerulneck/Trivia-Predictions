## 🎯 Critical Points

### 1. Execution Plan First
Each prompt requires Claude to output a **3-bullet execution plan** before any code. Review this plan before execution.

### 2. Bingo Border Color (CRITICAL)
- Must be: `#7dd3fc` (cool cyan)
- If wrong (warm orange): Re-run Prompt 3
- This color is the visual break between warm gameplay and cool structure

### 3. Atomic Tokens Only
- All colors: `var(--ht-*)`
- All sizes: tokens or semantic classes
- No hardcoding anywhere

### 4. Responsive Verification
Test at: 375px, 768px, 1024px, 1440px

### 5. Semantic Classes Mandatory
- `.ht-body` (paragraphs, not `text-base`)
- `.ht-eyebrow` (labels, not `text-xs`)
- `.ht-h1`, `.ht-h2` (headings, not arbitrary sizes)
