# Venue Game Card Rules (Saved)

These rules were provided on 2026-04-28 and should be treated as product requirements for upcoming implementation steps.

1. Keep exactly four visible game cards on Venue Home:
- Trivia
- Sports Bingo
- Pick 'Em
- Fantasy

2. Predictions must remain in the codebase but should not be displayed on the website right now.

3. Each visible game card should visually communicate its game identity through graphic design.

4. If a user has an active game, the game card should switch from static artwork to a miniature preview of that user's current game state.

5. Clicking a game card should smoothly expand that card into a full-screen experience, taking the user into that game where they left off.

6. Returning to Venue Home should use the reverse contraction animation, shrinking the full-screen game back into its corresponding game card preview.

7. These rules must be preserved for the next step: implementing game start screens that present rules before play starts.
